3. FE_XGBClassifier_GSCV¶

Kaggle score: 0.9617

GridSearchCV结果的表格分析：

{'max_depth': 3, 'scale_pos_weight': 100}

1. max_depth=4, scale_pos_weight依次增加：1、10、100。mean_train_score升高，mean_test_score降低。max_depth=3是特例，scale_pos_weight=10最高，比1略高，但是总体的趋势是确定的。因为本项目中，3是特例，scale_pos_weight=97~=100，所以，有待进一步测试scale_pos_weight超过100的结果，比如200，看是否会出现逆转。

2. scale_pos_weight=1，max_depth依次增加：3、4、5。mean_train_score升高，mean_test_score降低。

3. 从以上两条结论可以看出，max_depth和scale_pos_weight增加都增加了模型的复杂度，增加了对train_data的拟合能力，但是，泛化能力却下降了。

4. mean_fit_time和mean_score_time，随着max_depth的增加而升高，而scale_pos_weight的增加则不影响结果。

5. std_test_score和std_train_score值都很小，说明，KFold的各个split的数据分布比较一致。

6. std_fit_time值较大，说明模型间差异较大，下一步的Ensemble可能会有更好的结果？

7. 现在倾向于让max_depth=5，先过拟合，然后通过增加数据量的方式来增加泛化能力，如果后面数据量增加导致显存不够时，再考虑降低。有没有必要增大，现在还不确定。

8. val score: 0.9604，kaggle score: 0.9617，与TalkingdataAFD2018_XGBClassifier_GridSearchCV_20180427_152307_9825.html（kaggle score: 0.9666）相比，使用day6作为验证数据集比使用train_test_split的结果准确很多。也说明，数据的分布是比较均匀的。

def save_feature(x_data, y_data, file_name):
    print(y_data[:5])
    if os.path.exists(file_name):
        os.remove(file_name)
        print('File removed: %s' % file_name)
    with h5py.File(file_name) as h:
        h.create_dataset('x_data', data=x_data)
        h.create_dataset('y_data', data=y_data)
    print('File saved:   %s' % file_name)

def load_feature(file_name):
    with h5py.File(file_name, 'r') as h:
        x_data = np.array(h['x_data'])
        y_data = np.array(h['y_data'])
    print('File loaded:  %s' % file_name)
    print(y_data[:5])

def save_proba(y_train_proba, y_train, y_val_proba, y_val, y_test_proba, click_ids, file_name):
    print(click_ids[:5])
    if os.path.exists(file_name):
        os.remove(file_name)
        print('File removed: %s' % file_name)
    with h5py.File(file_name) as h:
        h.create_dataset('y_train_proba', data=y_train_proba)
        h.create_dataset('y_train', data=y_train)
        h.create_dataset('y_val_proba', data=y_val_proba)
        h.create_dataset('y_val', data=y_val)
        h.create_dataset('y_test_proba', data=y_test_proba)
        h.create_dataset('click_ids', data=click_ids)
    print('File saved:   %s' % file_name)

def load_proba(file_name):

        y_train_proba = np.array(h['y_train_proba'])

        click_ids = np.array(h['click_ids'])
    print('File loaded:  %s' % file_name)
    print(click_ids[:5])
    
    return y_train_proba, y_train, y_val_proba, y_val, y_test_proba, click_ids
